# 🎮 Hangman Game

A simple and interactive command-line Hangman game built using Python as part of my CodeAlpha Internship.

---

## 🚀 Features

* 🎯 Random word selection
* 💡 Hint system for each word
* ❌ Tracks wrong guesses
* ❤️ Limited attempts (6 tries)
* 🔁 Replay option after game ends
* ✅ Input validation for better user experience

---

## 🧠 Concepts Used

* Python basics
* Loops (`while`)
* Conditional statements (`if-else`)
* Functions
* Lists & dictionaries
* String handling

---

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the file:

```bash
python hangman_.py
```
